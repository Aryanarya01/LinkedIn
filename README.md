 # 🚀 ConnectSphere

A full-stack social networking platform inspired by LinkedIn, where users can build professional profiles, connect with others, share posts, and showcase their career journey.

---

## ✨ Features

* 👤 User Authentication (Login / Register)
* 🧑‍💼 Profile Management

  * Bio, Profile Picture
  * Work History
  * Education
* 🤝 Connection System (Send / Accept Requests)
* 📝 Create & View Posts
* 🖼️ Media Upload Support
* 🔍 View Other User Profiles
* 📄 **Resume Upload & Download**
* ⚡ Dynamic UI with real-time updates

---

## 🛠️ Tech Stack

**Frontend**

* Next.js
* React
* Redux Toolkit
* CSS Modules

**Backend**

* Node.js
* Express.js
* MongoDB
* JWT Authentication

---

## 📂 Project Structure

```
/client   → Frontend (Next.js)
/server   → Backend (Express + MongoDB)
```

---

## ⚙️ Installation

### 1️⃣ Clone the repo

```
git clone https://github.com/your-username/connectsphere.git
cd connectsphere
```

### 2️⃣ Install dependencies

**Frontend**

```
cd client
npm install
npm run dev
```

**Backend**

```
cd server
npm install
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file in the server folder:

```
PORT=9080
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
